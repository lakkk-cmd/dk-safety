# 대경안심전기 발표 대본

## 3분 버전 (핵심 브리핑)

안녕하세요.  
이번에 구축한 서비스는 `대경안심전기`의 온라인 예약/운영 시스템입니다.

핵심 컨셉은 `우리 집 전기 주치의`이며,  
고객은 메인 페이지에서 바로 예약 또는 문의로 진입할 수 있습니다.

예약 페이지에서는 이름, 연락처, 주소, 희망일, 요청사항을 입력하고 접수할 수 있고,  
전화번호 형식, 과거 날짜, 요청사항 길이 등 기본 검증을 적용해 잘못된 접수를 줄였습니다.

관리자 페이지에서는 로그인 후 예약을 검색/필터/정렬해서 빠르게 찾을 수 있고,  
상태를 `접수`, `진행중`, `완료`로 변경하면서 메모를 남길 수 있습니다.

운영 안정성을 위해 백업/복원 체계를 함께 구축했습니다.  
예약 추가 시 자동 스냅샷이 누적되고, 수동 백업도 가능하며, 복원 전 미리보기가 제공됩니다.  
복원 시에는 체크포인트를 자동 생성하고, 복원 결과를 diff로 보여주기 때문에 리스크를 낮췄습니다.

정리하면, 지금 버전은 내부 시연과 운영 테스트가 가능한 1차 완성 상태이며,  
운영값 확정 후 빠르게 오픈 가능한 단계입니다.

## 10분 버전 (상세 시연 대본)

### 1) 도입 (1분)

안녕하세요.  
오늘은 `대경안심전기`의 예약/운영 시스템 시연을 진행하겠습니다.

목표는 세 가지입니다.
- 고객 예약 동선 단순화
- 관리자 운영 효율화
- 백업/복원 기반의 안정성 확보

### 2) 고객 화면 시연 (3분)

먼저 메인 페이지입니다.

`우리 집 전기 주치의` 메시지를 전면에 두고,  
서비스 카드와 함께 예약/전화/카카오 상담으로 즉시 연결되도록 구성했습니다.

다음은 예약 페이지입니다.

정상 입력으로 접수하면 예약이 등록되고,  
잘못된 입력은 즉시 안내됩니다.
- 전화번호 형식 오류
- 과거 날짜 선택
- 요청사항 최소 길이 미달

이로써 접수 데이터 품질을 확보합니다.

### 3) 관리자 화면 시연 (3분)

관리자 로그인 후 예약 목록으로 이동합니다.

운영자는 검색/필터/정렬로 대상을 빠르게 찾고,  
상태를 `접수`, `진행중`, `완료`로 업데이트할 수 있습니다.

현장 메모를 저장하면 수정시각이 기록되어 운영 히스토리를 추적할 수 있습니다.

또한 CSV 다운로드로 외부 보고/백업 자료 정리가 가능합니다.

오른쪽 활동 로그 패널에서는 신규접수, 상태변경, 메모변경, 백업복원 이벤트를 확인할 수 있습니다.

### 4) 백업/복원 안정성 시연 (2분)

백업 패널에서 수동 백업 생성이 가능합니다.

스냅샷별 미리보기를 통해 복원 전 건수/기간을 확인하고,  
복원 시 체크포인트 라벨(복원 사유)을 입력합니다.

복원 직전에는 체크포인트가 자동 생성되며,  
복원 후에는 추가/삭제/변경/동일 건수를 diff로 보여줍니다.

문제 발생 시 `긴급 롤백`으로 최신 체크포인트를 빠르게 복원할 수 있습니다.

### 5) 마무리 (1분)

현재 상태는 내부 시연과 운영 검증이 가능한 1차 완성본입니다.

남은 오픈 마감은
- 실 연락처/카카오 링크 반영
- 관리자 비밀번호 운영값 교체
- 최종 체크리스트 통과
입니다.

승인 주시면 바로 오픈 단계로 전환하겠습니다.
