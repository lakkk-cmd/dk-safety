**결정사항**:
- 특허 출원번호를 `10-2026-0082817`로 최종 확정(대표님 직접 확인) — 이전 코드값
  `4-2026-034423-8`은 오기였음.
- 사업자등록번호 208-20-57629, 일반과세자 여부 둘 다 대표님이 확인 — `브랜드 프로필` 문서의
  "미검증" 항목 해소.
- `OPENROUTER_API_KEY` 확보 → `.env.local`에 추가, `knowledge_base` 동기화 실제 실행·검증 완료.

**변경된 파일**:
- `scripts/seed-knowledge-base.ts`(특허번호 정정), 프로덕션 `knowledge_base` 테이블 `patch4_basic`
  행 직접 UPDATE(임베딩은 재생성 안 됨 — 아래 미해결 참고), `brain/wiki/profile/회사-프로필.md`
  (특허번호/사업자번호/과세유형 전부 "확인된 사실"로 이동), `brain/raw/decisions/2026-확정-기술결정.md`
  (정정 이력 append).
- `brain/wiki/playbooks/블로그-발행-절차.md`, `영상-승인-절차.md` 신규 작성.
- `scripts/brain-sync.ts` 버그 2건 수정: (1) 매니페스트가 파일 단위로만 동기화 여부를 기록해서,
  OpenRouter 키가 나중에 추가돼도 이미 "동기화됨"으로 표시된 문서는 knowledge_base에 영영
  안 들어갈 뻔함 — 타겟별(`syncedTargets.knowledgeBase`/`knowledgeChunks`)로 추적하도록 수정.
  (2) `category` 추출이 `path.sep`(Windows에서 `\`)로 분리했는데 `relPath`는 이미 `/`로
  정규화돼 있어서 분리가 항상 실패, 45개 문서 전부 category가 "wiki"로 잘못 들어감 — `/`로
  분리하도록 수정 후 매니페스트 초기화하고 재동기화, category별 정상 분포(lessons 32/
  playbooks 2/profile 2/systems 9) 확인.

**알게 된 것**:
- 증분 동기화 스크립트를 짤 때 "무엇이 바뀌었나"(콘텐츠 해시)와 "어디까지 반영됐나"(타겟별
  상태)를 분리해서 추적하지 않으면, 나중에 조건(API 키 유무 등)이 바뀌었을 때 조용히 누락이
  생긴다 — 실행 후 실제 DB를 쿼리해서 증거를 확인하지 않았다면 몰랐을 버그.
- `knowledge_base.category`처럼 콘텐츠 자체가 아닌 메타데이터만 틀렸을 땐 해시가 안 바뀌어서
  일반적인 "변경분만 처리" 로직이 재동기화를 안 함 — 이런 종류의 버그는 매니페스트를 지우고
  전체 재동기화해야 해소된다.

**미해결**:
- `brain/wiki/playbooks/`에 블로그/영상 2개만 있음 — 다른 반복 업무(현장점검 알림 발송 등)
  플레이북은 아직 없음.

**추가 결정사항** (같은 날 이어서):
- `patch4_basic` 임베딩 재생성 요청을 받고 `seed-knowledge-base.ts` 그대로 실행하려다 **중단** —
  그 스크립트는 `knowledge_base` 테이블 전체를 삭제(`delete().neq(...)`)한 뒤 8개 청크만
  재삽입하는 구조인데, 실제 테이블엔 **3879개 행**이 있었음(KEC 규정, PDF 지식베이스 수천 청크,
  brain/wiki 45개 등). 그대로 실행했으면 대부분 삭제될 뻔함 — API 기본 페이지 제한(1000) 때문에
  처음엔 1000개로 보여서 위험도를 과소평가했었고, `count: "exact"`로 재확인해서 발견함.
  대신 `scripts/reembed-patent.ts`로 `patch4_basic` 행 하나만 targeted UPDATE — 다른 3878개는
  그대로 유지됨(재확인 완료).
- **교훈**: Supabase REST 조회는 기본 최대 1000행만 반환한다 — 테이블 전체 크기를 판단할 땐
  `count: "exact", head: true`로 별도 확인해야 함. 이번엔 이 착시 때문에 위험한 전체 삭제
  스크립트의 블라스트 레이디어스를 실제보다 훨씬 작게 잘못 알릴 뻔했음.
